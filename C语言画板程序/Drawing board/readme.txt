程序简要说明

1.程序名称：
你的画板
2.  作者：
胡亦璇（学号：2018211884）
3.开发目的：
旨在使用一系列简单易懂的操作提示，帮助用户完成最基本的绘图。
4.  程序功能概述：
该程序为一个简单的画板，用户可以根据提示进行简单的颜色选择与图案绘制。
5.功能模块：
    i.)文件存储/读取/清空
	ii.)功能菜单选择
	 （1.清屏；2.画图；3.退出；4.重画）
	iii.)自定义绘图
	 （本画板提供颜色：红色，黄色，绿色，蓝色）
     （可绘图案：圆，线段，填充矩形，填充多边形）
	iv.)历史图形查询
	 （即输入图案类型查询是否存在该种图形）
	v.)重绘历史图形
	 （即将以前所绘的图形重新绘制）
	vi.)输入合法性检验
	 （即检查用户输入内容是否符合程序可执行功能；若否，提示再次输入）
